# Movie Recommendation System

A supervised learning based movie recommendation system built using:

- Python
- Scikit-learn
- Random Forest Regressor
- NLP (TF-IDF)
- Streamlit
- TMDB Dataset
- MovieLens 100K Dataset

## Features

- Movie recommendation using cosine similarity
- Supervised learning model comparison
- Random Forest based rating prediction
- Real user ratings integration
- Interactive Streamlit UI

## Algorithms Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- KNN Regressor
- XGBoost Regressor

## Datasets Used

- TMDB 5000 Movies Dataset
- MovieLens 100K Dataset

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Run Locally

```bash
pip install -r requirements.txt
cd streamlit_app
